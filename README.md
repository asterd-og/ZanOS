# ZanOS
- A x86_64 operating system made from scratch by astrido!

# Features
- Paging
- APIC/LAPIC/IOAPIC
- Multiprocessing multitasking
- Lapic timer
- ATA PIO Driver
- EXT2 Driver (read-only only, write WIP)
- VFS Driver (WIP)
